## 🚀 Projects

### 🤖 [Barbershop Telegram Bot](https://github.com/NazarSikora/barbershop-bot)
> Booking system for a barbershop, fully built in Telegram

A full-featured booking bot with master selection, time slot management, and admin panel with real-time notifications.

**Key features:**
- FSM-based multi-step booking flow
- Inline keyboards for master & time selection
- Admin panel with booking notifications
- Double-booking prevention
- SQLite database via aiosqlite

`Python` `aiogram 3.x` `FSM` `SQLite` `aiosqlite`

---

### 📊 [OLX Price Monitor Bot](https://github.com/NazarSikora/olx-parser-bot)
> Track OLX listing prices directly from Telegram

A bot that monitors OLX listings and instantly shows whether a price has dropped, risen, or stayed the same — no manual checking needed.

**Key features:**
- Parses real OLX listing pages on demand
- Detects and highlights price changes
- Personal watchlist per user
- Handles errors gracefully — timeouts, deleted listings
- Async HTTP requests for non-blocking performance

`Python` `aiogram 3.x` `httpx` `BeautifulSoup4` `SQLite`
